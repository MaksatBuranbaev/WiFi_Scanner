# WiFi_Scanner

WiFi_Scanner - это приложение для сканирования доступных WiFi-сетей.

## Установка

1. Клонируйте репозиторий.
   ```bash
   git clone https://github.com/MaksatBuranbaev/WiFi_Scanner
## Установка

1. Клонируйте репозиторий.
2. Установите зависимости через NuGet:
   - `Microsoft.EntityFrameworkCore`
   - `Microsoft.EntityFrameworkCore.Design`
   - `Npgsql.EntityFrameworkCore.PostgreSQL`
   - `NativeWifi`

3. Настройте строку подключения в `WifiDbContext.cs`.

## Использование

1. Запустите приложение.
2. Нажмите кнопку "Сканировать" для поиска доступных WiFi-сетей.
3. Просмотрите список найденных сетей и их уровень сигнала.
4. Нажмите кнопку "Сохранить", чтобы сохранить найденные сети в базу данных.

## Структура проекта

- **ViewModels**: Логика представления и управление состоянием приложения.
- **Models**: Модели данных, представляющие WiFi-сети.
- **Data**: Контекст базы данных и настройка подключения.
- **Commands**: Реализация команд для управления действиями в приложении.
- **Views**: Представления и интерфейс пользователя.
